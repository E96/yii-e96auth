# How to use
* Положить в расширения
* Использовать вместе с <https://github.com/Nodge/yii-eauth/>
# composer.json
```
{
    "require" : {
        "e96/yii-e96auth" : "~1.1"
    },
    "repositories" : [
        {
            "type" : "vcs",
            "url" : "git@bitbucket.org:e96shop/yii-e96auth.git"
        }
    ]
}
```
